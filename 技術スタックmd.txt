# La Briller プロジェクト：現在の実装ステータス (2026-03-11)

## 🏗 技術スタック (Confirmed)
- **Framework:** Next.js 16 (App Router / Turbopack)
- **Core Library:** React 19
- **Styling:** Tailwind CSS v4 (globals.css によるテーマ変数制御)
- **Components:** Shadcn UI (Radix UI) & Custom Web Components (Vanilla JS)

- **Deployment:** Cloudflare Pages (Wranglerによる自動デプロイ)

## 🎨 デザイン & 実装済みのこだわり
### 1. ヒーローセクション (Helo)
- キャッチコピーを最新の「大切な天然歯を守り抜くこと。〜」に更新済み。
- プレミアムな動画背景とグラスモルフィズムなボタンスタイル。

### 2. メリットセクション (Merits)
- フォントを「Noto Sans JP (ゴシック)」に統一し、現代的な印象に。
- スクロールに合わせてバブルが下からフワッと浮かび上がる「時間差アニメーション」を実装。
- 上下の余白を詰め、スマホ表示時のリズム感を向上。

### 3. アクセスセクション (Access)
- 背景色をオリジナルのアイシーブルー (`#E6EDF2`) に完全一致。
- クリニックの横長ロゴ (`prfaitclinic_logo_yoko2-2.png`) を中央上部に配置。
- 診療テーブルのヘッダー色を落ち着いたブルー (`#88AFC6`) に設定。

### 4. ギャラリー & インスタ (Social)
- **Antigravity Gallery**: バニラJS製。実機のiPhoneでも滑らかにループするよう、SSR非互換問題をReactラッパーで解決済み。
- **Instagram Section**: 余白を詰め、すぐ下のQAセクションへの繋がりをスムーズに。

## 🛠 ユーティリティ (固有機能)
- **globals.css**: 共通の余白調整クラス (`.section-py`, `.section-mb-sm` など) を定義済み。
